# Portfolio Website

Portfolio project with a static frontend structure (HTML, CSS, JavaScript) and a PHP contact form.

## Contents

- `index.html`: Home page
- `script.js`: Page interactions
- `styles/`: CSS files for layout, components, and responsive design
- `assets/`: Images, icons, fonts, and favicon
- `contact.php`: Contact form handling
- `legal-notice.html` and `privacy-policy.html`: Legal pages
